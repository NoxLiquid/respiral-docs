# Введение

Это дизайн-документация контент-пака **Respiral** для Space Station 14.

Документация описывает **замысел** систем (зачем они, как меняется опыт игрока), в дополнение к
**реализации** в прототипах (`Resources/Prototypes/_Respiral/**`) игрового репозитория. Задача —
чтобы диздок и контент не расходились: страница описывает идею, таблицы связывают её с конкретными
`id` прототипов, а раздел «Реализация» на каждой странице даёт ссылки на YAML и список известных дыр.

## Как устроена документация

- **Профессии → [Обзор](./professions/overview.md)** — карта текущего состояния: департаменты,
  навыки, станки, цепочки материалов и найденные пробелы.
- **Профессии → [План диздока](./professions/design-plan.md)** — по какому шаблону описывать
  департаменты и профессии (адаптация шаблона [space-wizards/docs](https://github.com/space-wizards/docs)).
- **Департаменты** — по одной странице на департамент по этому шаблону. Первый заполненный пример —
  [Городские](./departments/craftsman.md).

## Соглашения

- Весь контент Respiral живёт под `_Respiral/`, `id` с префиксом `Medieval`.
- Изменённый исходный контент закомментирован edit комментариями.
